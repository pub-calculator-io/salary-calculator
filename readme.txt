This free salary calculator easily switches between hourly, bi-weekly, monthly, and yearly pay. Holidays and vacation days need special consideration and adjustments.

[https://www.calculator.io/salary-calculator/](https://www.calculator.io/salary-calculator/)

== Usage ==

Add the Salary Calculator shortcode to your page, post or sidebar:

`[ci_salary_calculator]`

Add the following code to your template where you would like the Salary Calculator to appear:

`<?php display_ci_salary_calculator(); ?>`

== Installation ==

1. Upload the Salary Calculator /ci_salary_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Salary Calculator plugin through the "Plugins" menu in WordPress.
